# 멋쟁이사자처럼 한국외대(서울) 11기 FE 운영진 - 자바스크립트 & 리액트 교안

## 소개
멋쟁이사자처럼 한국외대(서울) 11기 프론트엔드 운영진이 작성한 자바스크립트 및 리액트 교안입니다. 프론트엔드 개발에 관심 있는 학생들에게 자바스크립트와 리액트의 기본 개념부터 고급 주제까지 체계적으로 학습할 수 있도록 도움을 주고자 작성하였습니다.
